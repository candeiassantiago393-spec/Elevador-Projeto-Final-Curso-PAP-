# Teste 01 — Botões + LEDs Exteriores com Memória

## Descrição geral

Este teste foi desenvolvido no âmbito da PAP do projeto do elevador de 4 pisos, com o objetivo de validar a parte dos **pedidos exteriores** do sistema.

Nesta fase da montagem real, o objetivo não é testar o elevador completo, mas sim confirmar se os **botões exteriores de chamada** e os respetivos **LEDs de indicação** estão corretamente ligados ao Arduino Mega e a funcionar de acordo com a lógica esperada.

Este teste permite verificar se, ao carregar num botão exterior de um determinado piso, o LED correspondente liga e permanece ligado, simulando o comportamento real de um elevador, onde o pedido fica registado até ser atendido.
